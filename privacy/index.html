Privacy Policy

Introduction

Base64 Image Converter is an open-source tool hosted at mrlipx.github.io/base64-image-converter. We believe strongly in user privacy and radical transparency. This policy explains exactly how the tool handles your data.

Data Collection & Processing

Local processing only

  • All image conversions are processed entirely in your browser using JavaScript
  • No images or Base64 strings are ever uploaded to any server
  • No data leaves your device at any point during the conversion process
  • You can verify this by checking the browser Network tab — zero requests are made when converting

No storage

  • We do not store any images, Base64 data, or conversion history
  • There is no server-side processing of any kind
  • All data exists only temporarily in your browser's memory and is cleared when you navigate away

No personal information

  • We do not collect any personal information
  • No account creation or login is required or possible
  • No email addresses, names, or identifiers are collected

Open Source Commitment

This tool is fully open source. Every line of code that runs in your browser is publicly visible. Privacy isn't a promise — it's verifiable.

Source code

  • Licensed under the MIT License — free to use, modify, and redistribute
  • Available on GitHub: github.com/MrLiPx/base64-image-converter
  • You can fork and self-host your own instance at any time

Community

  • Discussions and feedback: GitHub Discussions
  • Bug reports and feature requests: GitHub Issues

Analytics & Cookies

Analytics

  • We use only GitHub Pages built-in traffic analytics — aggregated, anonymous page-view counts
  • No personal identifiers are collected
  • No third-party analytics scripts (e.g. Google Analytics, Mixpanel) are loaded

Cookies

  • We use only essential cookies required for the site to function normally
  • No tracking, advertising, or third-party cookies
  • No cookie consent banner is needed because we set no non-essential cookies

Security

  • All conversions happen locally — there is no server-side attack surface for your data
  • The site is served over HTTPS via GitHub Pages with industry-standard TLS encryption
  • Open-source code means any security issues can be identified and fixed by the community

Compliance

Because we collect no personal data and perform no server-side processing, the compliance footprint is minimal by design. This policy is designed to be compatible with GDPR, CCPA, and other applicable privacy regulations.
