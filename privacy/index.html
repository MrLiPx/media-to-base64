Privacy Policy

Last updated: January 9, 2024

Introduction

Base64 Image Converter is an open-source tool hosted at mrlipx.github.io/base64-image-converter. We believe strongly in user privacy and transparency. This policy explains how the tool handles your data.

Data Collection & Processing

Local Processing Only

  • All image conversions are processed entirely in your browser using JavaScript
  • No images or Base64 strings are ever uploaded to any server
  • No data leaves your device during the conversion process
  • You can verify this by reviewing our open-source code on GitHub

No Storage

  • We do not store any images, Base64 data, or conversion history
  • No server-side processing is involved whatsoever
  • All data exists only temporarily in your browser's memory and is cleared when you leave

No Personal Information

  • We do not collect any personal information
  • No account creation is required
  • No email addresses or identifiers are collected

Open Source Commitment

Transparency

  • Our code is fully open source under the MIT License
  • Available on GitHub: github.com/MrLiPx/base64-image-converter
  • You can review, fork, or contribute to the code at any time

Community

  • Discussions and feedback: GitHub Discussions
  • Bug reports and feature requests via GitHub Issues
  • Community contributions are welcome

Analytics & Cookies

Minimal Analytics

  • We use GitHub Pages built-in analytics only (aggregated, anonymous page views)
  • No personal identifiers are collected
  • No third-party analytics scripts (e.g. Google Analytics) are loaded

Cookies

  • We use only essential cookies required for the site to function
  • No tracking, advertising, or third-party cookies

Security

  • All conversions happen locally — end-to-end security for your data
  • The site is served over HTTPS via GitHub Pages with industry-standard TLS encryption
  • No server-side processing means no server-side risk

Compliance

This privacy policy is designed to comply with GDPR, CCPA, and other applicable privacy regulations. Because we collect no personal data and perform no server-side processing, the compliance footprint is minimal by design.
